Project Features

This project helps detect LPG gas leakage and automatically reduces gas concentration using an exhaust fan.

🔥 Gas Detection

MQ-2 sensor detects LPG, methane, and smoke.

🚨 Instant Alert

Buzzer warns people when gas leakage occurs.

🌬 Automatic Ventilation

Motor fan helps remove gas from the environment.
