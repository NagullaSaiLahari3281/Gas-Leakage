# 🔥 Automated LPG Leakage Detection and Gas Reduction System

## 📌 Overview
The **Automated LPG Leakage Detection and Gas Reduction System** is designed to detect gas leakage and automatically reduce gas concentration using an exhaust fan. This system enhances safety in homes, kitchens, and industries by preventing accidents like fire and explosions.

---

## 🚀 Features
- Detects LPG gas leakage using MQ-2 sensor  
- Provides alert using buzzer and LED  
- Automatically activates exhaust fan  
- Reduces gas concentration in real-time  
- Low-cost and easy to implement  
- Works without human intervention  

---

## 🛠️ Components Used
- Arduino UNO  
- MQ-2 Gas Sensor  
- DC Motor (Exhaust Fan)  
- Buzzer  
- LED  
- Breadboard  
- Jumper Wires  
- Power Supply  

---

## ⚙️ Working Principle
1. MQ-2 sensor continuously monitors gas levels  
2. Arduino reads sensor values (0–1023 range)  
3. If gas exceeds threshold:
   - LED turns ON  
   - Buzzer alerts  
   - Exhaust fan turns ON  
4. When gas level reduces:
   - System returns to normal state
